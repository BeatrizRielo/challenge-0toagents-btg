From Zero to Agents

Configure agents, prompts, instructions e skills do GitHub Copilot para gerar uma aplicação bancária completa.

⏱️ 1h 30min
👥 Até 5 pessoas
🤖 Apenas GitHub Copilot
🏆 100 pontos
⚠️ Não escreva código de aplicação manualmente
Configure os arquivos em .github/ e use o Copilot Chat (Agent mode) para gerar o backend.

🎯 Objetivo

Entregar uma API bancária funcional em Python (FastAPI) com CRUD de contas, transferências e relatório de segurança visível no frontend.

📋

CRUD Contas

Criar, listar, atualizar, excluir contas bancárias

💸

Transferências

Transferir saldo entre contas com validações

🔒

Relatório Segurança

Scan de vulnerabilidades com score no frontend

⚙️ Arquivos de Customização

Vocês vão corrigir 3 arquivos e criar 3 novos. Estes controlam como o Copilot entende o projeto.

📜

copilot-instructions.md

Regras globais — aplica em todos os arquivos automaticamente

📐

.instructions.md

Regras por tipo de arquivo via applyTo glob

🤖

.agent.md

Agente especializado — invocado com @nome no Chat

💬

.prompt.md

Prompt reutilizável com {{variáveis}} — via 📎 no Chat

🧠

SKILL.md

Conhecimento de domínio bancário — ativado automaticamente por contexto

📊 Fases

Fase 1 — Fundação

Fácil 30 min 30 pts
  • CORRIGIRcopilot-instructions.md — completar seções de segurança, testes e logs
  • CORRIGIRSKILL.md — adicionar regras de transferência, CPF e LGPD
  • CRIARpython-backend.instructions.md — regras para backend/**/*.py
  • GERARmodels.py + database.py — usar Copilot Agent mode

Fase 2 — Geração

Médio 30 min 40 pts
  • CRIARcreate-crud.prompt.md — prompt com {{entity_name}} que gera CRUD completo
  • CORRIGIRsecurity-reviewer.agent.md — tools erradas, falta output format e rules
  • GERARBackend completo — app.py, routes, services, health, swagger

Fase 3 — Segurança

Difícil 30 min 30 pts
  • CRIARquality-reviewer.agent.md — agent que roda testes e verifica coverage
  • GERARreport_service.py + routes — relatório de segurança com score
  • VALIDARFrontend — relatório visível em /pages/security-report.html

⭐ Bônus — Qualidade Total

Extra +20 pts

Terminou as 3 fases? Vá além e conquiste o título de Legend.

  • CRIARquality-guardian.agent.md — agent QA que roda testes e verifica coverage ≥ 80%
  • CRIARquality-report.prompt.md — prompt para executar testes e gerar relatório JSON
  • GERAREndpoint /api/reports/quality — retorna coverage, testes, files_analyzed
  • VALIDARTestes ≥ 80% + relatório visível em /pages/quality-report.html
